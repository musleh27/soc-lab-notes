# Windows Login Events (4624 & 4625)

## Event 4625 — Failed Login
- Description: Failed login attempt  
- What I did: Entered wrong password multiple times  
- What I saw: Event ID 4625 in Event Viewer  
- Why it matters: Can indicate brute force attempts  

---

## Event 4624 — Successful Login
- Description: Successful login  
- What I did: Logged in with correct password  
- What I saw: Event ID 4624  
- Why it matters: Normal activity but important after failed attempts  

---

## Summary
- Practiced generating failed and successful login events  
- Observed logs in Event Viewer  
- Learned difference between 4624 and 4625
